Рабочее место ЭТД предназначено для работы с электронными трудовыми договорами и дополнительными соглашениями к электронным трудовым договорам.

Рабочее место ЭТД состоит из трех разделов:

·         «Сведения об основных договорах».

Информация в раздел выводится на основании зарегистрированных в системе документов «Прием на работу», «Прием на работу списком», «Начальная штатная расстановка».

·         «Сведения о дополнительных соглашениях».

Информация в раздел выводится на основании документа «Кадровый перевод».

·         «Сотрудники к расторжению».

Информация в раздел выводится на основании документов «Увольнение», «Увольнение списком».

Список кадровых изменений для формирования дополнительных соглашений и список сотрудников к расторжению ЭТД формируются только по зарегистрированным в ЕСУТД договорам (отправленным в ЕСУТД договорам). Если договор ЭТД сотрудника не зарегистрирован в ЕСУТД (не отправлен) данные по кадровым изменениям такого сотрудника не отражается в рабочем месте ЭТД.

Разделы сгруппированы по действиям, которые необходимо произвести с ЭТД:

·         По ссылке «Все» выводится список всех электронных трудовых договоров, как уже созданных, так и тех которые необходимо создать;

·         По ссылке «Не созданы» отображается список сотрудников, по которым требуется создать ЭТД.

·         По ссылке «Не отправлены» отображается список ЭТД, которые были созданы, но еще не отправлены в ЕСУТД.

·         По ссылке «В ожидании ответа» отображается список ЭТД, которые были отправлены, но не обработаны в ЕСУТД.
