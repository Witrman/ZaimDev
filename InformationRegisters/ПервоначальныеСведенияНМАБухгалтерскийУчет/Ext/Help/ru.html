Регистр сведений "Первоначальные сведения о НМА (бухгалтерский учет)"


Регистр содержит первоначальные сведения о нематериальных активах организаций для целей бухгалтерского учета:

  • Регистратор;
  • Период;
  • Нематериальный актив;
  • Признак "Начислять амортизацию";
  • Объем продукции (работ) для начисления амортизации в натуральных ед.;
  • Первоначальная стоимость;
  • Способ начисления амортизации;
  • Способ поступления;
  • Срок полезного использования (мес.);
  • Стоимость для вычисления амортизации.


Данные из регистра используются для расчета амортизации объектов НМА.

По данным регистра определяется организация, в которой учитывается объект, в определенный момент времени.

Регистр является периодическим. Данные регистра изменяются документом "Принятие к учету НМА".
