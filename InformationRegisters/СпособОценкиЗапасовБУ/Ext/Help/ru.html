Регистр сведений «Способ оценки запасов»

Регистр предназначен для установки способа оценки запасов (ЛИФО, ФИФО, по средней). Для каждого счета учета ТМЗ может быть установлен свой способ оценки. В случае, если для счета не задано значение способа оценки, считается, что для данного счета установлен способ «по средней». Если установлен способ оценки «ЛИФО» или «ФИФО» для счета добавляется аналитика «Партия».
