Регистр сведений "Применение вычетов по ИПН"

Регистр сведений "Применение вычетов по ИПН" используется до 2018 года.


Регистр сведений является периодическим. В регистре хранятся сведения о том, в какой организации будет применен вычет по ИПН для физического лица.

В регистре указывается следующая информация:

  • Физическое лицо - сотрудник;
  • Налогоплательщик - организация, которая выступает в роли налогового агента по ИПН;
  • Дата – дата, с которой начнет применяться вычет по ИПН.
