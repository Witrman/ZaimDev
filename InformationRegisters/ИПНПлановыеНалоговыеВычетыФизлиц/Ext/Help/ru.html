Плановые налоговые вычеты физических лиц по ИПН (до 2018 года)

Регистр сведений «Плановые налоговые вычеты физических лиц по ИПН»

Регистр сведений «Плановые налоговые вычеты физических лиц по ИПН» используется до 2018 года.

Регистр сведений вызывается для ввода и редактирования информации из пункта «Главное меню» / «Все функции» / «Регистры сведений» / «Плановые налоговые вычеты физических лиц по ИПН».

В данном регистре хранятся сведения о плановых предоставляемых вычетах по ИПН в соответствии с пп. 4)-5) п. 1 ст. 166 Налогового Кодекса РК.

При определении дохода работника, облагаемого у источника выплаты, за каждый месяц в течение календарного года независимо от периодичности выплат, могут применяются следующие налоговые вычеты:

Регистр содержит следующую информацию.

Регистр содержит следующую информацию.
  • Период – дата регистрации заявления на предоставление вычета в базе.
  • Сотрудник – физическое лицо, подавшее заявление на предоставление вычета.
  • Налогоплательщик – организация, которая выступает в роли налогового агента по ИПН.
  • Месяц налогового периода – месяц налогового периода, в котором необходимо предоставить вычет (определяется по дате платежа, указанной в документе «Заявление на предоставление вычета по ИПН»).
  • Вычет ИПН - вид вычета из справочника «Вычеты ИПН».
  • Документ–основание – документ-заявление, на основании которого предоставляется вычет.
  • Сумма вычета - размер предоставляемого вычета.
  • сумма страховых премий, вносимых в свою пользу физическим лицом по договорам накопительного страхования;
  • суммы, направленные на погашение вознаграждения по займам, полученным физическим лицом-резидентом Республики Казахстан в жилищных строительных сберегательных банках на проведение мероприятий по улучшению жилищных условий на территории Республики Казахстан, в соответствии с законодательством Республики Казахстан о жилищных строительных сбережениях.
