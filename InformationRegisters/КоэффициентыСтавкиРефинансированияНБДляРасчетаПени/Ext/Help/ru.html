Регистр сведений "Коэффициенты ставки рефинансирования НБ для расчета пени"

Регистр сведений предназначен для хранения сведений о коэффициентах, применяемых в ставке рефинансирования Национального Банка Республики Казахстан при расчете пени за несвоевременное перечисление налогов, сборов, отчислений.

  • Организация. Организация, для которой следует применять указанный коэффициент. Если значение не заполнено, то соответствующий коэффициент будет применяться для всех организаций, для которых не заданы строки в регистре.
  • Вид налога. Вида налога, сбора, отчисления из справочника "Налоги, сборы, отчисления".
  • Коэффициент. Значение коэффициента.
