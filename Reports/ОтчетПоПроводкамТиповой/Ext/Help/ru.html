Отчет «Отчет по проводкам (бухгалтерский учет)»

Отчет «Отчет по проводкам (бухгалтерский учет)» показывает список проводок, отобранных по заданным условиям по бухгалтерскому учету. Каждая строка отчета соответствует одной корреспонденции счетов (проводке).

Автоматическое суммирование позволяет быстро получить сумму выделенных ячеек. Может быть выделен как диапазон ячеек, так и отдельные ячейки.

Из отчета можно открыть документ-регистратор, которым были записаны указанные проводки.

В отчете выводится следующая информация:

  • Период. Дата документа-регистратора, которым была сформирована данная проводка.
  • Документ. Сведения о документе и операции, которые сформировали проводку.
  • Аналитика Дт. Аналитика Кт. Аналитика по дебету и кредиту проводки.
  • Дебет. Указывается счет дебета и сумма проводки.
  • Кредит. Указывается счет кредита и сумма проводки.
  • Текущее сальдо. Сальдо выбранного счета, получившееся после записи проводки.
  • Итоговые суммы по проводкам по дебету и кредиту.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки».

  • Настройка аналитики. Позволяет выводить организацию или структурное подразделение в графе с аналитикой.
  • Отбор. Указывается реквизит, по которому будет происходить отбор данных в отчете, задается тип сравнения, а также указывается конкретное значение выбранного реквизита.
  • Показатели. В отчете могут отображаться данные бухгалтерского учета. Можно выводить данные о количестве и валютной сумме.
  • Оформление. Указываются настройки заголовка, вывод подписей, выделение отрицательных, вариант оформления отчета.
