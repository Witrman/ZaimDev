Отчет «Кассовая книга».

Отчет «Кассовая книга» формирует регламентную форму кассовой книги. Данный отчет формируется по организации и кассе предприятия, выбранных в настройках отчета за указанный период. При формировании кассовой книги номер сформированного листа запоминается и записывается в регистре сведений «Номера листов кассовой книги» с соответствующей датой.

В настройках также можно указать ряд параметров:

• Разбивать листы по дням. Если признак установлен, то данные за каждый день, в котором были какие-либо движения по кассе, будут выводиться на отдельном листе. Если признак не установлен - данные будут выводиться единым списком, упорядоченном по дате выписки первичных документов.

• Сортировать документы по виду. В пределах дня документы будут дополнительно упорядочены по виду.

• Сортировать документы по номеру. В переделах дня документы будут упорядочены по номеру.

• Номера документов по параметрам учета. При установке этого флага в графе «Номер документа» нумерация документов будет выводится согласно настройкам параметров учета.

• Выводить основания кассовых ордеров. При установке этого флажка в графе кассовой книги «От кого получено или кому выдано» дополнительно будет выводиться содержание полей «Основание» приходных и расходных кассовых ордеров.

• Выводить название кассы. При установленном признаке в заголовке отчета будет выводится наименование кассы.

• Печатать вкладной лист. При установленном признаке отчет содержит раздел «Вкладной лист кассовой книги».

• Печатать отчет кассира. При установленном признаке отчет содержит раздел «Отчет кассира».

• Сформировать титульный лист. Позволяет получить печатную форму титульного листа кассовой книги по итогам года.

• Пересчитать номера листов с начала года. Данный признак имеет смысл устанавливать при формировании отчета в целом за год. При этом, будет выполнена перенумерация листов кассовой книги в регистре сведений «Номера листов кассовой книги».

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки». Кнопка «Сохранить настройки» предназначена для сохранения заданных настроек. Кнопка «Выбрать настройки» предназначена для выбора ранее сохраненных настроек. Кнопка «Скрыть настройки» предназначена для закрытия окна настроек.

Кнопка «Печать» предназначена для формирования печатной формы отчета.
