Отчет «Структура задолженности организаций по ЕП».

Отчет «Структура задолженности организаций по ЕП» предоставляет возможность получить сведения об изменении накопленной задолженности по единому платежу сотрудников организаций перед Госкорпорацией за указанный в отчете период в следующих разрезах:

• Организация;
• Месяц образования задолженности;
• Вид платежа;
• Месяц;
• Сотрудник.

Помимо сумм задолженности на начало и конец периода в отчет можно вывести общие исчисленные и перечисленные суммы.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки». Кнопка «Сохранить настройки» предназначена для сохранения заданных настроек. Кнопка «Выбрать настройки» предназначена для выбора ранее сохраненных настроек. Кнопка «Скрыть настройки» предназначена для закрытия окна настроек.

Кнопка «Печать» предназначена для формирования печатной формы отчета.
