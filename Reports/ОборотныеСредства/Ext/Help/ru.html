Отчет «Оборотные средства»

Информация для руководителя

Отчет «Оборотные средства» предназначен для анализа оборотных средств организации. В отчете можно проанализировать удельный вес каждой статьи в общей величине оборотных средств на начало и конец периода.

Информация для бухгалтера

В отчете выводится информация по следующим показателям:

  • Денежные средства – рассчитывается итоговая сумма строк: наличные, безналичные и деньги в пути.
    • Наличные – сальдо по дебету счета 1010.
    • Безналичные – итоговая сумма сальдо по дебету счетов 1030, 1040, 1050, 1060, 1080, 1090.
    • Деньги в пути – сальдо по дебету счета 1020.
  • Краткосрочные финансовые вложения - сальдо по дебету счетов раздела 1100 (кроме 1170) - сальдо по кредиту счета 1170.
  • Дебиторская задолженность – рассчитывается итоговая сумма строк: задолженность покупателей, задолженность подотчетных лиц, прочая задолженность работников, прочие активы.
    • Задолженность покупателей - итоговая сумма сальдо по дебету счетов 1210, 1220, 1230, 1240, 1260, 1160, 1270, 1710, 1280.
    • Задолженность подотчетных лиц – сальдо по дебету счета 1251.
    • Прочая задолженность работников – итоговая сумма сальдо по дебету счетов 1252, 1253, 1254.
    • Прочие активы – итоговая сумма сальдо по дебету счетов 1720, 1730 и 1750.
  • Запасы – рассчитывается итоговая сумма строк: сырье и материалы, готовая продукция, товары, незавершенное производство, прочие запасы.
    • Сырье и материалы. Рассчитывается по следующей формуле: «Сальдо по дебету счета 1310» - «Сальдо по кредиту счету 1361».
    • Готовая продукция. Рассчитывается по следующей формуле: «Сальдо по дебету счета 1320» - «Сальдо по кредиту счета 1362».
    • Товары. Рассчитывается по следующей формуле: «Сальдо по дебету счета 1330» - «Сальдо по кредиту счету 1363».
    • Незавершенное производство – сальдо по дебету счетов раздела 1340.
    • Прочие запасы – сальдо по дебету счетов раздела 1350.

В отчет выводятся только статьи оборотных средств, по которым значения не равны нулю.

Из анализа данных можно исключить отдельные счета.
