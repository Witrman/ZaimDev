Справка-расчет "Резервы по вознаграждениям работникам"

Данный отчет предназначен для анализа результатов формирования резерва по вознаграждениям сотрудников. Размеры отчислений в резервы задаются в регистре сведений «Размеры отчислений в резервы»

В справочнике «Резервы» в табличной части «Базовые виды расчета» необходимо задать список видов начислений заработной платы, которые будут выступать в качестве базы для расчета ежемесячных отчислений в резерв.

На закладке «Размеры отчислений» следует указать проценты отчислений в резерв для будущих вознаграждений, а так же оплаты социального налога и социальных отчислений.

Ресурсы регистра:

  • % отчислений - процент отчислений в резерв по вознаграждениям работникам (на оплату отпуска, премии по итогам работы за год и т.п.) от суммы базовых видов начислений;
  • % СН - процент отчислений в резерв для покрытия затрат по социальному налогу, связанному с начислением вознаграждения работникам;
  • % СО - процент отчислений в резерв для покрытия затрат по социальном отчислениям, связанным с начислением вознаграждения работникам;
  • % ОППВ - процент отчислений в резерв для покрытия затрат по обязательным профессиональным пенсионным взносам;
  • % ОПВР - процент отчислений в резерв для покрытия затрат по обязательным пенсионным взносам работодателя;
  • % ООСМС - процент отчислений в резерв для покрытия затрат по взносам на обязательное социальное медицинское страхование;
  • Счет учета - счет учета резерва в бухгалтерском учете.

Настройки из регистра сведений «Размеры отчислений в резервы» используются при автозаполнении документа «Отражение зарплаты в регламентированном учете».

Сумма формируемого резерва определяется по следующим формулам:

  • Сумма резерва по вознаграждениям = Сумма базовых видов расчета * %отчислений;
  • Сумма резерва СН = Сумма резерва по вознаграждениям * %СН;
  • Сумма резерва СО = Сумма резерва по вознаграждениям * %СО;
  • Сумма резерва ОППВ = Сумма резерва по вознаграждениям * %ОППВ;
  • Сумма резерва ООСМС = Сумма резерва по вознаграждениям * %ООСМС;
  • Сумма резерва ОПВР = Сумма резерва по вознаграждениям * %ОПВР.
  • Общая сумма резерва = Сумма резерва по вознаграждениям + Сумма резерва СН + Сумма резерва СО + Сумма резерва ОППВ + Сумма резерва ООСМС + Сумма резерва ОПВР.
