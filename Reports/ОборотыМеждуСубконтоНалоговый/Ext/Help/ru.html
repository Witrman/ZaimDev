Отчет «Обороты между субконто (налоговый учет)»

Отчет «Обороты между субконто (бухгалтерский учет)» выводит список всех оборотов между списком выбранных субконто и списком корреспондирующих субконто. Виды субконто и виды корр. субконто можно указать как на форме отчета, так и в настройках отчета.

Отчет предоставляет пользователю вариант расшифровки. Двойной щелчок мыши по строки с записью счета позволяет получить расшифровку этого счета с помощью отчета «Отчет по проводкам (налоговый учет)».

В отчет выводится следующая информация:

  • Вид субконто.
  • Вид корр. субконто.
  • Дебет. Показывает дебетовую сумму между выбранными видами субконто и корр. субконто.
  • Кредит. Показывает кредитовую сумму между выбранными видами субконто и корр. субконто.

Для отчета может быть сформирована печатная форма.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки».

  • Виды субконто. Указываются виды субконто и виды корр. субконто, по которым формируется отчет.
  • Группировка. В отчете можно задать группировку по видам субконто и видам корр. субконто, указанным для формирования отчета. В списке указываются реквизиты, по которым необходимо сгруппировать данные отчета.
  • Отбор. Указывается реквизит, по которому будет происходить отбор данных в отчете, задается тип сравнения, а также указывается конкретное значение выбранного реквизита.
  • Показатели. В отчете могут отображаться данные бухгалтерского учета. Можно выводить данные о количестве и валютной сумме.
  • Дополнительные поля. Указываются дополнительные данные, которые необходимо вывести в отчете.
  • Сортировка. Данные отчета можно упорядочить по графам (полям) отчета.
  • Оформление. Указываются настройки заголовка, вывод подписей, выделение отрицательных, вариант оформления отчета.
