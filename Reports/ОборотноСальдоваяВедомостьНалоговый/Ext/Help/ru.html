Отчет «Оборотно-сальдовая ведомость (налоговый учет)».

Отчет «Оборотно-сальдовая ведомость (налоговый учет)» предназначен для формирования оборотно-сальдовой ведомости по всем счетам (субсчетам) налогового учета за определенный период времени.

Каждая строка отчета соответствует определенному счету или субсчету. Счета в отчете упорядочены в порядке возрастания.

Для каждого счета показаны суммы остатков на начало и на конец периода по дебету и кредиту, а также суммы оборотов за период.

Отчет предоставляет пользователю несколько вариантов расшифровки. Двойной щелчок мыши по строки с записью счета позволяет получить расшифровку этого счета с помощью другого отчета:

  • Отчет «Оборотно-сальдовая ведомость по счету (налоговый учет)»;
  • Отчет «Карточка счета (налоговый учет)»;
  • Отчет «Анализ счета (налоговый учет)»;
  • Отчет «Обороты счета (налоговый учет)».

В отчет выводится следующая информация:

  • Счет. Наименование. Указываются счета или субсчета, а также их наименования, по которым есть остаток или обороты за указанный период времени.
  • Сальдо на начало периода. Показывает дебетовое и кредитовое сальдо на начало периода по указанному счету.
  • Обороты за период. Показывает дебетовый и дебетовый оборот за период по указанному счету.
  • Сальдо на конец периода. Показывает дебетовое и кредитовое сальдо на конец периода по указанному счету.

Для отчета может быть сформирована печатная форма.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки».

  • Группировка. В отчете можно задать общую группировку по отчету и группировку счетов.
  • Развернутое сальдо. Указываются параметры для получения развернутого сальдо.
  • Отбор. Указывается реквизит, по которому будет происходить отбор данных в отчете, задается тип сравнения, а также указывается конкретное значение выбранного реквизита.
  • Показатели. В отчете могут отображаться данные бухгалтерского учета. Можно выводить данные о валютной сумме.
  • Дополнительные поля. Указываются дополнительные данные, которые необходимо вывести в отчете.
  • Оформление. Указываются настройки заголовка, вывод подписей, выделение отрицательных, вариант оформления отчета.
