Оборотно-сальдовая ведомость по счету (бух.)

Отчет «Оборотно-сальдовая ведомость по счету (бухгалтерский учет)».

Отчет «Оборотно-сальдовая ведомость по счету (бухгалтерский учет)».

Отчет «Оборотно-сальдовая ведомость по счету (бухгалтерский учет)» показывает начальные и конечные остатки, а также обороты за определенный период для выбранного счета по бухгалтерскому учету. Для счетов, по которым ведется аналитический учет, можно получить остатки и обороты с разбивкой по объектам аналитического учета. Если выбранный счет содержит субсчета, то отчет может быть сформирован с детализацией по субсчетам.

Отчет позволяет выводить развернутое сальдо для выбранного счета. При этом развернутое сальдо рассчитывается для каждого уровня группировки и по счету в целом.

Отчет предоставляет пользователю получить расшифровку выбранной ячейки с помощью отчета «Карточка счета (бухгалтерский учет)».

В отчет выводится следующая информация:

  • Счет. Аналитические объекты. Указывается счет и список аналитических объектов, в разрезе которых формируется отчет.
  • Сальдо на начало периода. Показывает дебетовое и кредитовое сальдо на начало периода по указанному счету.
  • Обороты за период. Показывает дебетовый и дебетовый оборот за период по указанному счету.
  • Сальдо на конец периода. Показывает дебетовое и кредитовое сальдо на конец периода по указанному счету.

Для отчета может быть сформирована печатная форма.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки».

  • Группировка. В отчете можно задать группировку по субсчетам и дополнительным полям. В поле «Периодичность» указывается вариант разбивки отчета по периодам времени.
  • Отбор. Указывается реквизит, по которому будет происходить отбор данных в отчете, задается тип сравнения, а также указывается конкретное значение выбранного реквизита.
  • Показатели. В отчете могут отображаться данные бухгалтерского учета. Можно выводить данные о количестве, валютной сумме и развернутом сальдо.
  • Дополнительные поля. Указываются дополнительные данные, которые необходимо вывести в отчете.
  • Сортировка. Данные отчета можно упорядочить по графам (полям) отчета.
  • Оформление. Указываются настройки заголовка, вывод подписей, выделение отрицательных, вариант оформления отчета.
