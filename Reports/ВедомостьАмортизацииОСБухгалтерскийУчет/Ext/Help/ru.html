Отчет «Ведомость амортизации ОС за период (бухгалтерский учет)» предназначен для анализа данных по основным средствам.

В ведомости начисления амортизации можно проанализировать:

- перечень основных средств в разрезе МОЛ и Подразделений;
- стоимость и амортизацию по ОС (первоначальная, изменения и на конец периода).

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки». Кнопка «Сохранить настройки» предназначена для сохранения заданных настроек. Кнопка «Выбрать настройки» предназначена для выбора ранее сохраненных настроек.
Кнопка «Скрыть настройки» предназначена для закрытия окна настроек.

Параметры панелей настроек:

• Группировка. С помощью панели группировки можно задать группировку по объектам. Порядок группировки изменяется с помощью кнопок «Вверх», «Вниз».

• Отбор. В колонке «Поле» следует выбрать параметр или реквизит, доступный для настройки отчета. В последней колонке указывается конкретное значение выбранного параметра или реквизита параметра. Если в табличной части указать два или более условия заполнения нескольких строк, то при отборе данных для формирования отчета будет проверяться одновременное соблюдение всех указанных условий.

• Показатели. Указываются показатели, которые будут участвовать в формировании отчета. При необходимости пользователь может самостоятельно настроить видимость перечисленных показателей.

• Дополнительные поля. Указываются дополнительные данные, которые необходимо вывести в отчете.

• Сортировка. С помощью панели сортировки можно упорядочить значения по графам (полям) отчета.

• Оформление. С помощью панели оформления можно управлять настройками заголовка, выводом подписей и другими. Также можно выбрать вариант цветовой схемы оформления отчета.

Кнопка «Печать» предназначена для формирования печатной формы отчета.
