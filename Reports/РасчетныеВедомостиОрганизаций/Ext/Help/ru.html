Отчет «Расчетная ведомость организации»

Отчет «Расчетная ведомость организации» предоставляет возможность получить сведения о произведенных расчетах по заработной плате организации за указанный в отчете период в разрезе отдельных видов начислений, удержаний, налогов, взносов и отчислений.

В отчете сотрудники объединены в группы по подразделениям, что дает пользователю возможность управлять отображением конкретных групп.

Отчет можно сформировать за определенный период, с детализацией по организации, подразделению или сотруднику.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки». Кнопка «Сохранить настройки» предназначена для сохранения заданных настроек. Кнопка «Выбрать настройки» предназначена для выбора ранее сохраненных настроек. Кнопка «Скрыть настройки» предназначена для закрытия окна настроек.

• Группировка. В отчете можно задать группировку строк по субконто и по группам видов расчета.
• Отбор. Указывается реквизит, по которому будет происходить отбор данных в отчете, задается вид сравнения, а также указывается конкретное значение выбранного реквизита.
• Дополнительные поля. Указываются дополнительные данные, которые необходимо вывести в отчете.
• Сортировка. Данные отчета можно упорядочить по графам (полям) отчета.
• Оформление. Указываются настройки заголовка, вывод подписей, выделение отрицательных, вариант оформления отчета.

Кнопка «Печать» предназначена для формирования печатной формы отчета.
