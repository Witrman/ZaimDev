Отчет «Основные настройки организации»

Данный отчет показывает сводную информацию по основным настройкам выбранной организации – учетные политики, контактная информация, ответственные лица и т.д. Отчет вызывается по кнопке «Настройка организации» из формы организации.

В настройках отчета выбираются разделы, по которым нужно выводить информацию. Предусмотрены следующие разделы:

  • Общие сведения. Выводится информация о наименовании организации и ее регистрационных данных (РНН, БИН/ИИН и т.д.).
  • Основной банковский счет. Указываются данные основного банковского счета организации.
  • Учетная политика (бух. учет). Выводится информация об основных настройках учетной политики по бухгалтерскому учету.
  • Учетная политика (нал. учет). Выводится информация об основных настройках учетной политики по налоговому учету.
  • Учетная политика по персоналу. Выводится информация об основных настройках учетной политики по персоналу.
  • Контактная информация. Выводятся контактные данные организации.
  • Ответственные лица. Указывается информация обо всех ответственных лицах организации.
  • Общие настройки. Указываются данные аналитического учета ТМЗ и способа оценки запасов.
