Отчет «Отчет по движению стоимости ФА (НУ)»

Отчет «Отчет по движению стоимости ФА (НУ)» позволяет получить данные о движении стоимостных балансов групп фиксированный активов в налоговом учете.

Отчет позволяет просмотреть регламентные показатели в виде расчетных величин, вычисляемых непосредственно в отчете, а так же суммы, фактически отраженные в регистре налогового учета по всем фиксированным активам.

Ведомость может быть построена по данным налогового учета выбранной организации.

В отчет выводится следующая информация:

  • Группа фиксированного актива. Указывается группа фиксированного актива в налоговом учете.
  • Фиксированный актив.
  • Стоимость на начало. Стоимость фиксированного актива на начала указанного периода.
  • Поступление. Стоимость при поступлении фиксированного актива в течение указанного периода.
  • Увеличение стоимости за счет ремонта.
  • Выбытие. Стоимость при выбытии (продаже или списания) фиксированного актива.
  • Перемещение.
  • Стоимость до корректировок. Рассчитывается автоматически по формуле: Начальный остаток + Поступление + Увеличение стоимости за счет ремонта – Выбытие – Перемещение.
  • Предельная норма амортизации, %. Указывается предельная норма амортизации из регистра сведений «Сведения о группах фиксированных активов».
  • Применяемая норма амортизации, %. Указывается применяемая норма амортизации из регистра сведений «Сведения о группах фиксированных активов».
  • Амортизация (учетная). Сумма амортизации основного средства.
  • Текущий ремонт. Указывается сумма ремонта по виду ремонта «Текущий».
  • Ремонт (Преференции). Указывается сумма ремонта по виду ремонта «Преференции».
  • Капитальный ремонт. Указывается сумма ремонта по виду ремонта «Капитальный».
  • Вычет на ремонт, %. Указывается вычет на ремонт из регистра сведений «Сведения о группах фиксированных активов».
  • Списание в пределах минимума (НУ).
  • Списание при выбытии всех активов.
  • Доход от превышения стоимости выбывших ФА.
  • Стоимость на конец.

Для отчета может быть сформирована печатная форма.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки».

  • Группировка. В отчете можно задать группировку по субсчетам и дополнительным полям.
  • Показатели. Можно настроить отображение различных показателей, информацию по которым необходимо видеть в отчете.
  • Отбор. Указывается реквизит, по которому будет происходить отбор данных в отчете, задается тип сравнения, а также указывается конкретное значение выбранного реквизита.
  • Сортировка. Данные отчета можно упорядочить по графам (полям) отчета.
  • Оформление. Указываются настройки заголовка, вывод подписей, выделение отрицательных, вариант оформления отчета.
