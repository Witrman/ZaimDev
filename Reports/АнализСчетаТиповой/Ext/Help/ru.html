Отчет "Анализ счета (бухгалтерский учет)".

Отчет «Анализ счета (бухгалтерский учет)» отображает данных бухгалтерского учета по оборотам между указанным счетом и всеми остальными счетами за определенный период, а также остатки по выбранному счету на начало и конец периода. Отчет показывает в каком объеме и в корреспонденции с какими счетами поступали и списывались средства на выбранном счете. По содержанию выводимой информации данный отчет похож на отчет «Обороты счета (бухгалтерский учет)», отличие заключается, главным образом, в форме предоставления информации.

Данные отчета можно вывести с разбивкой по периодам времени: за период отчета, по дням, по месяцам и т.д.

Отчет можно сформировать с детализацией по субсчетам или по объектам аналитического учета (субконто).

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки». Кнопка «Сохранить настройки» предназначена для сохранения заданных настроек. Кнопка «Выбрать настройки» предназначена для выбора ранее сохраненных настроек. Кнопка «Скрыть настройки» предназначена для закрытия окна настроек.

  • Группировка. В отчете можно задать группировку по субконто. При установке признака «По субсчетам» задается дополнительная детализация по субсчетам. В списке указываются реквизиты, по которым необходимо сгруппировать данные отчета. В поле «Периодичность» указывается вариант разбивки отчета по периодам времени.
  • Отбор. Указывается реквизит, по которому будет происходить отбор данных в отчете, задается тип сравнения, а также указывается конкретное значение выбранного реквизита.
  • Показатели. В отчете могут отображаться данные бухгалтерского учета. Можно выводить данные о количестве и валютной сумме, а также данные развернутого сальдо.
  • Дополнительные поля. Указываются дополнительные данные, которые необходимо вывести в отчете.
  • Сортировка. Данные отчета можно упорядочить по графам (полям) отчета.
  • Оформление. Указываются настройки заголовка, вывод подписей, выделение отрицательных, вариант оформления отчета.

Кнопка «Печать» предназначена для формирования печатной формы отчета.
Поле «Сумма» предназначено для суммирования данных выделенных ячеек.
