Отчет «Обороты счета (бухгалтерский учет)»

Отчет «Обороты счета (бухгалтерский учет)» показывает информацию о начальном и конечном остатке, оборотах выбранного счета за указанный период времени и детализацию этих оборотов в разрезе корреспондирующих счетов. То есть, отчет показывает, в каком объеме и в корреспонденции с какими счетами поступали и списывались средства на выбранном счете в течение указанного периода времени. Отчет можно использовать как компактную выборку информации, относящейся к выбранному счету, из отчетов «Оборотно-сальдовая ведомость (бухгалтерский учет)» и «Шахматная ведомость».

В отчете можно вывести развернутое сальдо для выбранного счета. При этом развернутое сальдо рассчитывается для каждого уровня группировки и по счету в целом.

Автоматическое суммирование позволяет быстро получить сумму выделенных ячеек. Может быть выделен как диапазон ячеек, так и отдельные ячейки.

Для отчета предусмотрена возможность детализации (расшифровки) записи отчетом «Обороты счета (бухгалтерский учет)».

В отчете выводится следующая информация:

  • Начальное сальдо (дебетовое и кредитовое). Показывает начальное сальдо по указанному счету и уровню группировки.
  • Оборот Дт. Показывает суммарный дебетовый оборот счета.
  • Корреспондирующий счет. Разбивка дебетового оборота счета по корреспондирующим счетам (т.е. обороты с дебета выбранного счета в кредиты корреспондирующих счетов).
  • Оборот Кт. Показывает суммарный кредитовый оборот.
  • Корреспондирующий счет. Разбивка кредитового оборота счета по корреспондирующим счетам (т.е. обороты с кредита выбранного счета в дебеты корреспондирующих счетов).

Для отчета может быть сформирована печатная форма.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки».

  • Группировка. В отчете можно задать группировку по субконто по субсчетам корр. счетов, при установке которых задается дополнительная детализация. В списке указываются реквизиты, по которым необходимо сгруппировать данные отчета. В поле «Периодичность» указывается вариант разбивки отчета по периодам времени.
  • Отбор. Указывается реквизит, по которому будет происходить отбор данных в отчете, задается тип сравнения, а также указывается конкретное значение выбранного реквизита.
  • Показатели. В отчете могут отображаться данные бухгалтерского учета. Можно выводить данные о количестве и валютной сумме. Для отчета предназначены настройка отображения выводимых данных.
  • Дополнительные поля. Указываются дополнительные данные, которые необходимо вывести в отчете.
  • Сортировка. Данные отчета можно упорядочить по графам (полям) отчета.
  • Оформление. Указываются настройки заголовка, вывод подписей, выделение отрицательных, вариант оформления отчета.
