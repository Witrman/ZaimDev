Отчет «Универсальный Регистр Налогового Учета»

Отчет предназначен для формирования печатной формы регистров налогового учета по произвольно выбранному разделу налогового учета (соответствующему регистру накопления).

В настройках отчета необходимо указать следующие реквизиты:

  • Период – период, за который необходимо сформировать отчет.
  • Налогоплательщик – организация, по которой строится отчет, а также список его структурных подразделений, если в информационной базе поддерживается работа со структурными подразделениями.
  • Таблица - существует возможность вывести в отчет:
    • Обороты - сводные данные по регистру;
    • Движения первичных документов - детальные записи первичных документов.
  • Разделы налогового учета, по которым формируется отчет:
    • Акциз
    • Взаиморасчеты организаций с контрагентами-физлицами
    • Взаиморасчеты организаций с получателями по ИЛ
    • Взаиморасчеты с бюджетом по НДС с измененным сроком уплаты
    • Взаиморасчеты с бюджетом по НДС, подлежащему уплате за нерезидента
    • Взаиморасчеты с депонентами организаций
    • Взаиморасчеты с сотрудниками организаций
    • Выплаченные доходы сотрудникам организаций по налоговому учету
    • ИПН расчеты с бюджетом
    • ИПН сведения о доходах
    • Корректировка НДС
    • Корректировка НДС к возмещению
    • Налог на добавленную стоимость
    • НДС к возмещению
    • НДС, взаиморасчеты с бюджетом
    • ОПВ подлежит перечислению в фонды
    • ОПВ расчеты с фондами
    • ОПВ сведения о доходах
    • СН исчисленный
    • СН сведения о доходах
    • СО расчеты с фондами
    • СО сведения о доходах


Отчет поддерживает возможность произвольной настройки выводимых показателей и группировок, однажды сделанные настройки могут быть сохранены и затем восстановлены для повторного использования.

Результат отчета может быть сохранен, а также распечатан.
