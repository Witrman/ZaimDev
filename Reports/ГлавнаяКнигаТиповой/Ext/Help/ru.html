Отчет «Главная книга типовой»

Отчет позволяет вывести обороты счетов в корреспонденции со счетами и сальдо на начало и конец периода с заданной периодичностью.

Главная книга строится по всем счетам бухгалтерского учета для выбранной организации.

Реквизиты
  • Период - необходимо указать период, за который формируется отчет.
  • Организация - можно указать как одну, так и несколько организаций; по данным бухгалтерского учета указанных организаций строится отчет.

Действия
  • Кнопка Сформировать - по кнопке выполняется построение отчета в соответствии с текущей настройкой.
  • Кнопка Показать настройки - предназначена для редактирования перечня текущих настроек отчета; в открывшейся форме выполняется формирование условий отбора для построения отчета; список может содержать дополнительные параметры.
  • Кнопки Сохранить настройки, Выбрать настройки - позволяют сохранять пользовательские настройки отчета, чтобы использовать их в дальнейшем.

Закладка Общие
  • Выводить данные
    • По группам счетов – для детализации отчета по группам счетов.
    • По субсчетам – если флаг выставлен, будет произведен расчет развернутого сальдо по субсчетам.
    • По группам кор.счетов – для детализации отчета по группам кор.счетов.
    • По субсчетам кор.счетов - для детализации данных по субсчетам корр.счетов.
  • Периодичность отчета
    • Период - задает периодичность, с которой будут выводиться промежуточные итоги.
    • Все периоды - если флажок установлен, будут выводиться и те периоды, за которые не было оборота.
    • Разбивать по листам - при печати отчета данные каждого счета выводились в виде отдельного листа главной книги.
    • Выводить заголовок - позволяет получить более компактный вид отчета, скрывая заголовок ведомости. Кнопки сохранения и восстановления значений позволяют сохранять пользовательские настройки отчета, чтобы использовать их в дальнейшем.
    • Выводить подписи – при включенном признаке в печатной форме выводятся подписи ответственных лиц организации.
