Отчет «Расчетные листки организаций»

Отчет «Расчетные листки организаций» предоставляет возможность получить сведения о произведенных расчетах по заработной плате организации за указанный в отчете период в разрезе отдельных видов начислений, удержаний, налогов, взносов и отчислений, суммах выплаченной заработной платы и задолженности по заработной плате на начало и конец расчетного месяца для каждого сотрудника.

Отчет предоставляет возможность сформировать расчетные листки сотрудников организаций за произвольный период времени, отобранные и сгруппированные по различным критериям.

Отчет по умолчанию настроен так, чтобы в результате получились расчетные листки за текущий месяц, сгруппированные по организациям и их подразделениям.

Возможные группировки отчета:

• Организация;
• Подразделение;
• Сотрудник;
• Месяц начисления.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки». Кнопка «Сохранить настройки» предназначена для сохранения заданных настроек. Кнопка «Выбрать настройки» предназначена для выбора ранее сохраненных настроек. Кнопка «Скрыть настройки» предназначена для закрытия окна настроек.

Кнопка «Печать» предназначена для формирования печатной формы отчета.
