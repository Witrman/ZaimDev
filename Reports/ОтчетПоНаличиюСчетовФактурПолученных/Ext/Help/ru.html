Отчет может формироваться в двух вариантах:
Первый вариант – «Наличие счетов-фактур».
В данном варианте для документов поступления отражаются созданные на их основании счета-фактуры.
Второй вариант - «Наличие документов продажи».
В данном варианте - для счетов-фактур отражаются созданные на их основании документы поступления.
При выборе этого варианта изменяется состав группировок отчета - добавляется группировка «Счет-фактура».
По итогам показателей этой группировки можно определить, все ли документы поступления были созданы, а также совпадают ли суммы данных документов и данных счета-фактуры.

В отчете данные сгруппированы по "Данным документа продажи" и "Данным счета-фактуры"

В главной форме отчета можно задать следующие параметры отчета:

• Даты начала и окончания отчетного периода.

• Наименование организации, для бухгалтерского учета которой формируется отчет.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки».

В табличной части настроек отчета выбираются необходимые документы. В зависимости от значения параметра «Отбор» отчет формируется либо на основании документов, указанных в табличной части, либо на основании всех документов, кроме указанных в табличной части. Если список документов не заполнен, отчет формируется по всем документам с учетом остальных ограничений (организация, период и т.п.).

Результаты формирования отчета представляет собой таблицу со следующими полями:

Для варианта «Наличие счетов-фактур»:

• Документ-основание. Наименование документа.

• Счет-фактура. Наименование счета-фактуры, сформированного на основании данного документа.

• Проведен СФ. Признак проведения счета-фактуры.

Для варианта «Наличие документов отгрузки»:


• Счет-фактура

• Документ отгрузки. Наименование документа, сформированного на основании счет-фактуры.

Кнопка «Печать» предназначена для формирования печатной формы отчета.
