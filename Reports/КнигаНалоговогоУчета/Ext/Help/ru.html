Книга налогового учета

Отчет «Книга налогового учета

».

Отчет «Книга налогового учета» заполняется индивидуальными предпринимателями на упрощенной декларации, которые являются  плательщиками НДС.

В поле «Организация» доступны организации у которых в Учетной политике по НУ установлен признак «Плательщик НДС» и выбран режим «Упрощенная декларация», в карточке организации в поле «Юр./физ. лицо» заполнено значением «Физ. лицо».

Данные для заполнения Титульного листа берутся из карточки организации.

Ведомость учета доходов заполняется следующим образом:

  • Обороты по реализации - данные регистров накопления «НДС» и «Корректировка НДС»;
  • Обороты по счетам-фактурам, выставленным поставщиками -данные регистров накопления «НДС к возмещению»  и «Корректировка НДС к возмещению».

В настройках отчета устанавливается дата начала и окончания деятельности. 

Если в настройках отчета установлен признак  «Выводить итоги (месяц, квартал, полугодие) за весь период», в этом случае отчет будет сформирован с подведением итога за каждый день, месяц, квартал, полугодие, год, в зависимости от выбранного периода. Если данный признак будет снят, тогда отчет будет сформирован за указанный период без подведения промежуточных итогов за месяц, квартал. полугодие.  Период можно выбрать в рамках одного года.

Например: Признак установлен и выбран период 01.01.22 г.- 31.03.22 г., тогда будут подведены итоги за каждый день, каждый месяц и квартал. Если признак отключен, тогда промежуточные итоги за январь и февраль выводиться не будут.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки». Кнопка «Сохранить настройки» предназначена для сохранения заданных настроек. Кнопка «Выбрать настройки» предназначена для выбора ранее сохраненных настроек. Кнопка «Скрыть настройки» предназначена для закрытия окна настроек.


Кнопка «Печать» предназначена для формирования печатной формы отчета.
