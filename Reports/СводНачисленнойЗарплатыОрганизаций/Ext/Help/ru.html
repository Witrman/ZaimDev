Отчет «Свод начислений и удержаний организаций».

Отчет «Свод начислений и удержаний организаций» предоставляет возможность получить сводные данные о произведенных расчетах по заработной плате организации за указанный в отчете период в разрезе отдельных видов начислений, удержаний, налогов, взносов и отчислений.

Если установлен флажок «Сводные данные по организациям», тогда формируется один общий отчет по всем отобранным данным, иначе - по каждой организации формируется отдельный свод расчетов.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки». Кнопка «Сохранить настройки» предназначена для сохранения заданных настроек. Кнопка «Выбрать настройки» предназначена для выбора ранее сохраненных настроек. Кнопка «Скрыть настройки» предназначена для закрытия окна настроек.

Кнопка «Печать» предназначена для формирования печатной формы отчета.
