Отчет «Отчет по данным контрагента»

Отчет «Отчет по данным контрагента» показывает информацию, зарегистрированную для контрагента. Отчет может быть сформирован для определенного контрагента, для группы контрагентов или по всем контрагентам справочника (если поле «Контрагент» отчета не заполнено).

В отчете выводится следующая информация:

  • Общие данные контрагента.
  • Контактная информация контрагента.
  • Контактные лица контрагента.
  • Прочие свойства и значения контрагента.
