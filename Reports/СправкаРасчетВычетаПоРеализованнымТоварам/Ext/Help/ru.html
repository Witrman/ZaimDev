Отчет «Справка Расчет Вычета По Реализованным Товарам»

Справка-расчет формируется после проведения регламентной операции «Расчет вычета по реализованным товарам».

Данные СНД (БУ), СКД (БУ), Сумма приход (БУ), Сумма приход (ПР), Сумма расход (НУ) в отчете формируются и хранятся при проведении регламентной операции, а не рассчитываются на момент формирования отчета!!! Если после проведения регламентной операции были операции по поступлению/списании номенклатуры, то для достоверных данных необходимо повторно выполнить регламентную операцию «Расчет вычета по реализованным товарам»!

Отчет содержит следующие основные показатели.

Отчет содержит следующие основные показатели.
    • Сумма приход (ПР) – сумма поступивших товаров не в целях СГД, зафиксированная непосредственно в документах поступления.
    • Сумма расход (НУ) – сумма признания в НУ товаров, поступивших в не целях СГД, образуется при проведении регламентной операции «Расчет вычета по реализованным товарам» документа «Закрытие месяца».
    • Сумма расход (ВР) – отнесение всей суммы расхода (НУ) на вид учета ВР образуется при проведении регламентной операции «Расчет вычета по реализованным товарам» документа «Закрытие месяца».
    • СНД (БУ)- начальное дебетовое сальдо (по счетам запасов - 1310, 1320, 1330, 1351, 2932, по номенклатурным позициям, по которым в заданном периоде было списание продукции с видом учета ПР).
    • СКД (БУ) - конечное дебетовое сальдо (по счетам запасов - 1310, 1320, 1330, 1351, 2932, по номенклатурным позициям, по которым в заданном периоде было списание продукции с видом учета ПР).
    • Сумма оборот Дт (БУ) - общая сумма поступления товаров по данным бухгалтерского учета (по счетам запасов - 1310, 1320, 1330, 1351, 2932, по номенклатурным позициям, по которым в заданном периоде было списание продукции с видом учета ПР), до выполнения операции «Расчет вычета по реализованным товарам».
    • Сумма оборот Дт (ПР) - сумма поступления товаров с видом учета ПР по данным налогового учета (по счетам запасов - 1310Н, 1320Н, 1330Н, 1351Н, по номенклатурным позициям, по которым в заданном периоде было списание продукции с видом учета ПР).
    • Сумма оборот Кт (НУ) - сумма реализованных товаров по виду учета НУ (по счетам запасов - 1310Н, 1320Н, 1330Н, 1351Н, по номенклатурным позициям, по которым в заданном периоде было списание продукции с видом учета НУ), до выполнения операции «Расчет вычета по реализованным товарам».
    • Сумма расход (ПР) – сумма реализованных товаров, которые числятся с видом учета ПР, зафиксированная в документах реализации/списания товаров.
    • Сумма расход (НУ) – сумма признания в НУ товаров, реализованных ПР. Определяется соотношением общей суммы уменьшения вычета к сумме расхода ПР. Образуется при проведении регламентной операции «Расчет вычета по реализованным товарам» документа «Закрытие месяца».
    • Сумма расход (ВР) – уменьшения суммы, числящийся по виду учета ПР на сумму уменьшения вычета по НУ, образуется при проведении регламентной операции «Расчет вычета по реализованным товарам» документа «Закрытие месяца».

Для редактирования перечня текущих настроек отчета выполните команду «Показать настройки». В открывшейся форме выполняется формирование условий отбора для построения отчета. Список может содержать дополнительные параметры.

Результат отчета может быть сохранен, а также распечатан.

  • Номенклатура – Номенклатурная позиция по которой либо было поступление не в целях СГД (вид учета = ПР), либо выполнялось списание данной номенклатуры.
  • Поступление товаров не в целях СГД - группа показателей для товаров, поступивших не в целях СГД:
  • Данные для расчета вычета (данные по БУ и НУ до выполнения регламентной операции) - группа выводится, только при наличии списания ТМЗ, не в целях СГД, в указанном периоде.
  • Распределение общей суммы уменьшения вычета ПР пропорционально сумме расхода(ПР) - группа выводится, только при наличии списания ТМЗ, не в целях СГД, в указанном периоде. Расчет суммы уменьшения вычета ПР с распределением по реализованной номенклатуре (номенклатура, которая поступала не в целях СГД, с видом учета ПР).

Вычет - сумма вычета для декларации, которая должна получиться после проведения регламентной операции "Расчет вычета по реализованным товарам". Определяется по формуле: Вычет= СНД (БУ) – СКД (БУ) +Оборот Дт (БУ) - Оборот Дт (ПР)
