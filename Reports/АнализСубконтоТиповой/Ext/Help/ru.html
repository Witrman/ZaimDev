Отчет "Анализ субконто (бухгалтерский)"

Отчет "Анализ субконто (бухгалтерский)" предназначен для предоставления данных бухгалтерского учета по указанному виду субконто: начальное и конечное сальдо, обороты за период по счетам. Отчет можно сформировать не только по выбранному субконто или нескольким субконто, но и по значению субконто или реквизиту значения субконто. Для каждого значения субконто предусматривается детализация по счетам.

Данные можно вывести с разбивкой по периодам времени: за период отчета, по дням, по месяцам и т.д.

В отчете можно вывести развернутое сальдо. При этом развернутое сальдо рассчитывается для каждого уровня группировки.

Кнопка "Печать" предназначена для формирования печатной формы отчета.

Поле "Сумма" предназначено для суммирования данных выделенных ячеек.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку "Показать настройки". Кнопка "Сохранить настройки" предназначена для сохранения заданных настроек. Кнопка "Выбрать настройки" предназначена для выбора ранее сохраненных настроек. Кнопка "Скрыть настройки" предназначена для закрытия окна настроек.

  • Виды субконто. Указывается список субконто, по которым необходимо сформировать отчет.
  • Группировка. В отчете можно задать группировку по субконто. При установке признака "По субсчетам" задается дополнительная детализация по субсчетам. В списке указываются реквизиты, по которым необходимо сгруппировать данные отчета. В поле "Периодичность" выбирается вариант разбивки отчета по периодам времени.
  • Отбор. Указывается реквизит, по которому будет произведен отбор в отчете, задается тип сравнения, а также указывается конкретное значение выбранного реквизита.
  • Показатели. В отчете могут отображаться данные бухгалтерского учета. Можно выводить данные о количестве и валютной сумме.
  • Дополнительные поля. Указываются дополнительные данные, которые необходимо вывести в отчете.
  • Сортировка. Данные отчета можно упорядочить по графам (полям) отчета.
  • Оформление. Указываются настройки заголовка, вывод подписей, выделение отрицательных, вариант оформления отчета.
