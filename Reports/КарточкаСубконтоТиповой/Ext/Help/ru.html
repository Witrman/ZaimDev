Отчет «Карточка субконто (бухгалтерский учет)»

Отчет «Карточка субконто (бухгалтерский учет)» предназначен для предоставления упорядоченной информации по датам выборки корреспонденции счетов, которые относятся к выбранному периоду времени и в которых был использован указанный вид субконто. По структуре отчет аналогичен отчету «Карточка счета (бухгалтерский учет)».

Каждая строка отчета соответствует одной корреспонденции счетов. В отчет выводится итоговая информация: начальный остаток и конечный остаток по выбранному счету, а также итоговые обороты.

Данные отчета можно вывести с разбивкой по периодам времени: за период отчета, по дням, по месяцам и т.д.

Настройки отчета задаются в отдельном окне, которое открывается при нажатии на кнопку «Показать настройки». Для сохранения заданных настроек предназначена кнопка «Сохранить настройки». Кнопка «Выбрать настройки» позволяет выбрать ранее сохраненные настройки. Кнопка «Скрыть настройки» предназначена для закрытия окна настроек.

  • Виды субконто. Указывается список субконто, по которым необходимо сформировать отчет. В поле «Периодичность» указывается вариант разбивки отчета по периодам времени.
  • Отбор. Указывается реквизит, по которому будет происходить отбор данных в отчете, задается тип сравнения, а также указывается конкретное значение выбранного реквизита.
  • Показатели. В отчете могут отображаться данные бухгалтерского учета. Можно выводить данные о количестве и валютной сумме.
  • Оформление. Указываются настройки заголовка, вывод подписей, выделение отрицательных, вариант оформления отчета.

Кнопка «Печать» предназначена для формирования печатной формы отчета.
Поле «Сумма» предназначено для суммирования данных выделенных ячеек.
